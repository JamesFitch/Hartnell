How to add a Campus or Community location

The location content type is used predominantly through out the site.

To create a location types:

  1. Click the Add Content button, for this tutorial we will be creating a Location.
  2. In the vertical tabs located in the left hand side, we are going to be completing the Standard Fields region.
  4. Your mandatory areas are Title, Location Type but the more complete this information is the better the chance is students/future students will contact you.
  5. To successfully list this content type you will need to at least complete the Title, Address and Phone Number fields on the Location content type.
  6. To make sure it appears on this directory check the Published box and of course always click the Save button to preserve your changes.
