How to create and add testimonials to your flexible page

The Testimonials block allows you to share testimonials with your site visitors. They are filtered by a few different views and displayed on the flexible page block in a teaser-style block. You may display testimonials of students, faculty, community members, or a random selection of all testimonials.

How to create testimonials

  1. At the top of the screen you should see a black admin menu bar and the gray add content menu bar customized for this site. In the Add content menu please select "Add Content".
  2. From the choices of content types you are going to choose "Testimonial".
  3. Here is a breakdown of the fields associated with this content type:
    • Title Every piece of content in Drupal has a title, and so does every page. In this one instance it is ok to leave this area blank.
    • Testimonial Image When displayed by the view this image will be resized to 140x160 pixels. The original content will show the image at original size so be careful not to upload an image too small or too large.
    • Attribution Here you should enter the name of person who made the quote.
    • Affiliation This field allows for the person's relationship to the school. For example: Hartnell College Alumni, current student, etc.
    • Testimonial Type You may select students, staff, or community.
    • Academic Area The program this person is affiliated with, this allows for this testimonial to appear on pages with related content. For example: culinary student testimonials will appear on culinary program related pages.
    • Body This area is for the actual testimonial due to limited space this area allows for up to 200 characters (including spaces).

How this works:

This display can be filtered placing following view pane:

  • View: Testimonials: Related Testimonials Shortlist Allows you to customize the list of testimonials by Academic areas, Testimonial Type and number of testimonials to show on this page; if no related testimonials are available this block will not show.


You can place, drag and drop this view to anywhere in panelizer. If there are no related testimonials this area will be blank.
