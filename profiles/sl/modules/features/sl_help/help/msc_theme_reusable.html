Reusable styles

Since they are based on CSS classes, these styles can be used repeatedly on a single page, as well as chained together to achieve a number of effects. For example, you could apply a left float with a color box or just the color box by itself.

Some of these styles utilize CSS3, so be aware not every browser will render the effects exactly the same, particularly rounded corners and drop-shadows.

CSS definitions for each of these styles can be found (and modified if necessary - though be careful!) in the stylesheet named "reusable-styles.scss" in the theme CSS directory (e.g. /sites/all/themes/omega_maricopa/css/reusable-styles.scss). Please note that we are using SASS/Compass to compile the CSS.

Use the documentation below to see examples of the styles and how to use them.

How to apply reusable styles

Panel panes

  1. Navigate to "Content" under "Panelizer."
  2. Click on the gear icon of the pane you want to apply the style to, then click on "CSS properties."
  3. In the field labeled "CSS class" put the reusable style class name, without any preceding period. If you put in more than one class name, separate each one with a space.
  4. Click "Save" and then be sure to click "Update and save" on the main page.

Boxes / blocks

  1. To apply a reusable style to a block or box, get to its configuration page. You can do so by
    • Going to Admin menu > Structure > Blocks, then finding the block/box in the list. Click its "Configure" link.

      Or
    • Clicking on its contextual link (the gear icon), then clicking "Configure block" or "Edit box."

  2. Find the "Advanced Settings" fieldset, then put your class names into the "Additional CSS classes" field.

Rich text editor

  1. While editing with the rich text editor, select the portion of your content you wish to apply the style to.
  2. Find the "Styles" drop-down input.
  3. Depending on what kind of element you select, you will have a variety of options.

Markup

  1. Make sure your rich text editor is switched off so it doesn't mess up your code.
  2. Put your class name as the value in a class attribute for whichever element you want to affect.
    <p>Lorem ipsum dolor sit amet, <span class="classname">consectetuer adipiscing elit</span>. Morbi commodo, ipsum sed pharetra gravida, orci magna rhoncus neque, id pulvinar odio lorem non turpis.</p>
  3. Note: You might not see a preview of your style directly in the rich text editor after switching back to the normal rich text editing. If you need to preview, click the "Preview" button at the bottom of the page.

Views

  1. When editing your view, make sure the "Advanced" fieldset is expanded.
  2. Click on the value next to "CSS class".
  3. Enter your class name into the field that appears.
  4. Depending on your purposes, you may want to change whether it applies to all the current view's displays or just this one.
  5. Click on "Apply" (or "Apply (this display)").
  6. Be sure to save your view.
