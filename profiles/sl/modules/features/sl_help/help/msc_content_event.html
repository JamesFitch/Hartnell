Adding an event

  1. Click on "Add content" in the admin menu.
  2. Click on "Event"
  3. In "Event Name" enter the name or title of the event.

  4. In the "Banner Image" field, browse to the image you would like to display at the top of the page and then click the "Upload" button.

    Note that your image will be resized and then cropped to 915x300 pixels regardless of its original dimensions. So you will want to upload an image that has those dimensions already or is larger.

  5. "Event Date" has several options. First, decide if you want the event to have an ending date. If not, uncheck "Show End Date." The "to:" option below will disappear.

    To choose the day, click on the "Date" field.

    A little calendar will pop up, allowing you to choose the correct day.

    To choose the time, click on the "Time" field.

    You may have to click twice to activate it. You can use the keyboard arrows to choose the time, or you can select each element (hour, minute, am/pm) and type it in.

    You can make an event repeating by clicking the "Repeat checkbox.

    You can set up your event so it repeats daily, weekly, monthly, or annually; you can set up which day or days it repeats on, and how many times the event occurs (or an end date.) You can also exclude or include dates.

  6. For "Location Name" choose the building where the event will be held, as well as any other relevant location information.
  7. In "Address" enter the street address of the event. This will both display as a human readable address as well as get turned into a Google Maps link.
  8. In "Event Description" input the description of the event. This is a rich text field so you can add various styles and insert images and videos. To see how to use rich text, visit this help page: Using the rich text editor.
  9. Under "Audience" choose the appropriate audience for this event.
  10. Under "Event Type" select the appropriate event type.
  11. Unless instructed otherwise, do not enter anything into "Tags."

  12. "Related Links" allows you to create a pseudo menu of related pages, either internal to the website or external on other websites.

    Enter the link text into "Title" and the relative path or URL into the "URL" field.

    To add multiple links, click "Add another item."

    Note: If you are linking to an internal page, it is best to use the node ID, which is the unique identifier of the piece of content. This prevents broken links if the human-readable path to the page changes. To find the node ID of a piece of content:

    1. Navigate to the page you want to link to (it can be helpful to do this in a separate browser tab or window).
    2. Ensure you are logged in as a content editor.
    3. Click on the content's "Edit," "Edit draft" or "New draft" tab.
    4. In the URL, find where it says "node/[somenumber]" e.g. "node/139".
    5. The number is the node ID. Copy that portion of the URL (e.g. "node/139") and put that into the URL field of the related link.
  13. In "Event Description" input the description of the event. This is a rich text field so you can add various styles and insert images and videos. To see how to use rich text, visit this help page: Using the rich text editor.
  14. Unless otherwise instructed, do not use the "Contact Person" field.
  15. The "Sign Up Form Link" allows you to direct visitors toward a signup page, either a webform in this site or an external webpage. The field works similarly to Related Links except you can only enter one sign up link and it will display as a button at the top of the page.

  16. If you are updating an event, before saving go to the "Revision information" fieldset.

    Ensure the "Create new revision" option is checked and then provide a brief explanation of your changes.

  17. When you have finished adding or updating the content, press "Save" at the bottom of the page.
  18. Note that the "Preview" functionality is limited and will not completely display how your content will look once published.
