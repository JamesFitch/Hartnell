Manage users

Creating a user

On this site, annonymous visitors can apply for an account, but administrator approval is required before the account can become active. Only site aministrators can assing roles.

To create a user:

  1. In the admin menu, go to "People."
  2. Click "Add user."
  3. In "Username" enter the name you want to give to the user. While spaces are allowed, it can be easier and simpler to make it all one word. A good pattern is lastname+first initial. E.g. smithj
  4. In "E-mail address" enter the user's email address. This should be an active email account, since it can be used if the user forgets their password.
  5. In "Password" and "Confirm password" enter in a password for the user. This can be anything you like, although you should write it down or otherwise find some way to let the person whose account it is know what their password is. For security's sake they should change it once they log in for the first time.
  6. Under "Roles" select which additional roles, if any, this user should have.
  7. Choose whether or not to "Notify user of new account." This is useful in helping new users learn their login info.
  8. Click "Create new account" to complete the new user account creation.

Editing a user's settings

To make changes to a user's settings:

  1. In the admin menu, go to "People."
  2. Find the user in the list. You can filter by role, and/or you can change the sorting to narrow your options.

  3. Once you find the user in the list, click on their "edit" link.
  4. Make your changes, then click "Save.
  5. If you want to cancel their account, click "Cancel account."

Making bulk edits/updates to users

You can edit more than one user at a time using the "Update options" feature.

  1. For each user you want to edit, check the box next to their username.
  2. After you have used the checkboxes to select all the users you want to edit, use the dropdown option under "Update options" to choose what you would like to change.
  3. Click "Update" to make your changes.

How roles work

Roles are sets of permissions that can be applied to individual users. Users can belong to more than one role. Two roles, authenticated users (those users that sign up for an account) and anonymous users (those either without an account or not logged in), are supplied by default with Drupal installations. This site also has the following additional roles:

  • Content creator/editor
  • Content reviewer/publisher
  • Section administrator
  • Site administrator
