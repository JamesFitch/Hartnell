Changing the social media icons

The images for the social media icons are stored in the theme directory, specifically: /sites/all/themes/omega_maricopa/images/ and are prefixed with "social-."

Changing an icon

If you need to change an icon, save your new version with the exact same name as the old one, then upload it to the images directory, overwriting the old file.

Try to keep the height approximately the same as the old version, to prevent any layout issues in the buttons.

Adding a new icon

This is more complex than just adding a new icon image; you will also need to create a new menu item and corresponding styles. That is beyond the scope of this help documentation.
