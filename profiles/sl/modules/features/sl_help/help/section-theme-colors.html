Overview

Site sections can be configured to display alternate colors in certain display elements.

Themeable elements include:

  • Title bar text and background colors.
  • Sidebar menu text and background colors.

Color options

Title Bar

  Text White Gold Gray Black
  Background Olive Blue Burgundy Gold

Sidebar Menu

  Text Gold White Gray Black
  Background Olive Gold Burgundy Blue

Setting Section Theme colors

To specify theme colors for a particular site section (including, by default, all sub-sections beneath it):

  • Find the Site Section term in the Site Section vocabulary listing.
  • Follow the Edit link, and scroll down to the "Theme Settings" section (if necessary, click "Theme Settings" in the vertical tabs at the left).
  • Under "Section Theme Colors", select "Custom" in the "Color Settings" dropdown list.
  • Specify a color for each themeable element.
  • Save changes.

Note that these colors will be not be applied to pages in child sections that have their "Color Settings" that override these settings (see "Sub-Section Overrides", below).

Inheriting Section Theme colors from parent Section

To have pages within a sub-section use the theme colors of its parent Section:

  • Find the Site Section term in the Site Section vocabulary listing.
  • Follow the Edit link, and scroll down to the "Theme Settings" section (if necessary, click "Theme Settings" in the vertical tabs at the left).
  • Under "Section Theme Colors", select "Inherit from parent section." in the "Color Settings" dropdown list.
  • Save changes.

Note that these colors will be not be applied to pages in child sections that have their "Color Settings" that override these settings (see "Sub-Section Overrides", below).

Sub-Section Overrides

Color settings for a section that's a parent of other sections will be not be applied to pages in any child sections that have their "Color Settings" set to either:

  • Custom. Pages within that section will use the colors specified for that section.
  • Default. Pages within that section will use the site's default cololrs.

In other words, settings for child sections (other than "Inherit") override the settings of parent sections.
