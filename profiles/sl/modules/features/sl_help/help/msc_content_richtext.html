Using the rich text editor (CKEditor)

When in a rich text field, you can apply a variety of styling to the text you input. Here are some items of note:

Bolding and italicizing:

Select some text then click the bold or italicize buttons to apply the styles.

Ordered and unordered lists:

Create a list separated by returns (the enter key), then select the list and click on either the ordered or unordered list buttons to apply the markup that will generate either bullets or numbers.

The tables button will help you create a table.

When creating tables:

  • If you want a table header, be sure to specifically set the header to the first row
  • Set the width to nothing. This will allow the table to fill up the available page width.
  • To edit the properties of a table after you've created it, right click on the table (while in the editor) and choose "Table Properties."

If you want to create a link

Select the text that should link, then click the link button.

A window will pop up where you can select if this is an internal link (within this website) or an external link (linking to another website).

If it is internal, start typing the title of the page in the "Link" field and it will autocomplete for you. If it's external, copy and paste the URL of the webpage into the "URL" field.

If you wish to remove a link from some text

Highlight the text then click the unlink button.

The embed media button

Allows you to paste embed code (such as from a video hosting service) into the page.

The security settings for this feature allow almost all "embed" code, but it limits all "iframe" markup to just YouTube and Vimeo. (Iframes are the current default embed code for those video services.)

To embed YouTube/Vimeo videos:
  1. First, get the embed code from either YouTube or Vimeo.

    For YouTube:

    1. Go to the video's page, then find the "Share" button.
    2. Click the "Embed" button to get the code to appear.
    3. Optionally choose custom embed options.
    4. Copy the iframe code to your clipboard.

    For Vimeo:

    1. Go to the video's page, then find the "Share" button.
    2. On the screen that appears, optionally choose custom embed options.

    3. Copy the iframe code to your clipboard.
  2. Once you have your iframe code, go back to the page you're editing/creating and click on the "Embed Media" button.
  3. In the window that pops up, paste your iframe code.
  4. Click "OK."

The IMCE button

Allows you to upload and insert images into the body of the page.

  1. First place your cursor where you want the image. Then, when you click on the IMCE button, a new window will open. You can choose from one of the already-uploaded images or you can upload a new one.
  2. To upload a new one, click on the "Upload" button in the top left.

    Choose your file and if you'd like to make any preset thumbnails, then click the "Upload" button.

    (You'll need to click the "X" at the top to get rid of the upload floating window.)
  3. Click the "Insert file" button to insert the image into the text.

Format

This lets you structure the page in terms of headings, paragraphs, and divs. For example, if you want your page to have several sections with subheadings, highlight your headings and then use the Format options to choose "Heading 2" or "Heading 3."

Styles

This allows you to add theme-generated styles to the text.

These are contextually based, meaning different styles will appear based on what element you have selected. For example, different options will appear for images than will appear for headings.

To create a button:
  1. Create a link in its own paragraph (use the link button)
  2. Highlight/select the link you just created, then use the Styles drop-down to find the button you want to use. The highlighted text will turn into a button.


  3. If you wish to add an icon, click the "Source" button then add the appropriate class to the anchor tag. See the Reusable Styles help page for a list of available button icons and their classes (the list appears at the bottom of the page).

Size:

Use this option to change the text size. If you are creating a section heading, however, do not make the text bigger but use the formatting options to give it the proper markup instead.

Erase formatting

If you wish to completely remove all formatting from a piece of text, highlight it and then click the eraser button.

Source:

For advanced users. Use this to view and modify the underlying markup of the body text.

This is particularly useful if you need to add an icon to a button. To add icons to buttons:

  1. Create a button using the Styles drop-down.
  2. Click on the "Source" button in the rich text editor
  3. Find your button's text, then add the class of your icon to the anchor tag. For example, the markup might look like this: <a class="button-light-gray" href="/maricopa/node/131">Test light gray button with icon</a> You will want to add your class after where it says button-light-gray. For example: <a class="button-light-gray phone" href="/maricopa/node/131">Test light gray button with icon</a> Be sure there is a space after the button class and that the icon class is inside of the quote marks. The class name will match the icon image name. To see available options, look at the Reusable Styles help page and/or look at the icons available in /sites/all/themes/omega_maricopa/images/button-icons
