Overview

The Flexible Page banner image display is handled via a custom Panel Pane, the "Dynamic Page Banner". This Pane has been added to each of the Flexible Page defaults, or it can manually be added to a Panel. The Dynamic Page Banner pane/block is located under "Custom blocks" in the Panelizer panel's "Add Content" pop-up window.

This feature dynamically chooses what banner image to display: either the banner attached to the individual page or the banner attached to the page's "Site Section".

How the Banner Image is selected

The following rules are used to determine what (if any) banner to display:

  • A banner attached to the page itself (i.e., with the Images->Banner Image field) will always be displayed.
  • If a banner image has not been set on the page, the system will look for a banner attached to the page's Site Section (i.e., with the About this content->Site Section field).
  • If no banner image has been set for the page's Site Section, the system will search upwards along the Site Section taxonomy tree. For instance, if a page's Site Section is "Men's Footbal" and the Site Section taxonomy tree looks like this:
    • Student Life
      • Athletics
        • Men's Football
    the system will first check for a banner image attached to "Men's Football", then "Athletics", and, finally, "Student Life".
  • If no banner image is found on the page, its Site Section or any ancestors thereof, no banner will be displayed.

Creating a page-specific Banner image

On the page's edit screen, select a Banner Image (on the Images tab). This will override any Banner supplied by this page's Site Section.

Creating a Site-Section-specific Banner image

To specify a default Banner image for a particular site section (including all sub-sections beneath it):

  • Find the Site Section term in the Site Section vocabulary listing.
  • Follow the Edit link and select a Banner Image (on the Theme Settings tab).

Note that this Banner will be not be applied to pages in this section wherever any of the following conditions is true:

  • A sub-section of this section has a banner image attached.
  • A page within this section has a Banner Image attached to it directly.
  • A page within this section has the "Hide Banner" box checked (see below).

Creating a page with no banner

In order to create a page with no banner, even when it is in a Site Section for which a Banner has been specified:

  • Go to the "Content" sub-tab of the "Panelizer" tab for the page.
  • Click the Settings Gear icon on the "Dynamic Banner" pane, and choose "Settings".
  • Place a check in the "Hide Banner" checkbox (in the "Configure" popup window).
  • Click "Finish" on the Configure window, then "Save" on the "Content" tab.
