The "tabs" are the links on the righthand side of the block as displayed on the Home Page (as shown in the screenshot below).

Each tab is linked to a particular "Audience" taxonomy term.

The Audience for each tab is set on the Content tab of the Audience Promotion Mini Panel. Each instance of "View: Audience Promotion Blocks" has an Audience assigned, under settings (click on the gear icon to the right of the pane title.)
