Managing menus

How to add a page to the main menu

You can add a page to the main menu one of two ways: as you're creating the page or through editing the menu directly. Either way, you'll first want to create an Editorial Site Section for the menu item.

Creating a Site Section term

  1. Starting in the admin menu, go to "Structure" -> "Taxonomy."
  2. Find "Editorial Site Section" and click on its "add terms" link.
  3. Enter in the Name of the new term. It should match what the menu item will be.
  4. Click on the "Relations" fieldset link.
  5. Under "Parent terms" find the parent item and select it.
  6. In most cases you can ignore the rest of the fields.
  7. Click "Save."
  8. Click on the "List" tab at the top of the page.

    Then find your new term and ensure it's got the right parent and placement. You may need to go to the next page if it does not appear on the first one. To adjust the order and nesting of a term item, click and drag its crosshairs icon, which appears to the left of its title.
  9. The next step is to add a new menu item and associate it with that term. Continue reading to learn how to do that while making a page or by directly editing the main menu.

Add a menu item when creating or editing a page

  1. While on the content edit page, find the "Menu settings" fieldset tab.
  2. Click on the checkbox for "Provide a menu link."
  3. Change or fill out the "Menu link title" field as desired. This is the text that people will see and click on in the menu.
  4. Under "Parent item," choose the parent item for this page.
  5. "Weight" helps determine the order of the menu items. Lower numbers will display the item higher; higher numbers will display the item lower in the menu. If this is confusing, you can change the menu items' order by editing the menu directly (see instructions below).
  6. You will likely not need to add anything to the rest of the fields.
  7. Find the "Site section" field (either under "Standard fields" or "About this content") and make sure it matches with the parent(s) of the menu item. This ensures proper editorial access.
  8. Remember to click "Save" to save your changes.
  9. The last step is to associate the new menu item with its Editorial Site Section. To do so:
    1. Using the admin toolbar, go to Structure -> Menus
    2. Find the menu you want to add the page to. Usually this will be "Main menu."
    3. Click on "list links" in the menu's row.
    4. Find your newly added menu item and click on its "edit" link.
    5. Find the "Power menu" fieldset, which allows you to sync menu items up with the Editorial Site Section. Under "Taxonomy plugin" find the matching term you created (as per the instructions above) and select it. Most of the other terms will be grayed out, indicating they have already been paired with a menu item.
    6. Click "Save."

Add a menu item through the menu

  1. Using the admin toolbar, go to Structure -> Menus
  2. Find the menu you want to add the page to. Usually this will be "Main menu."
  3. Click on "add link" in the menu's row.
  4. In "Menu link title" choose the display title for the menu item (this is what people will see and click on).
  5. In the "Path" field, enter the path to the menu item.

    Note: If you are linking to an internal page, it is best to use the node ID, which is the unique identifier of the piece of content. This prevents broken links if the human-readable path to the page changes. To find the node ID of a piece of content:

    1. Navigate to the page you want to link to (it can be helpful to do this in a separate browser tab or window).
    2. Ensure you are logged in as a content editor.
    3. Click on the content's "Edit," "Edit draft" or "New draft" tab.
    4. In the URL, find where it says "node/[somenumber]" e.g. "node/139".
    5. The number is the node ID. Copy that portion of the URL (e.g. "node/139") and put that into the URL field of the related link.
  6. Leave "Enabled" and "Show as expanded" at their defaults. You can uncheck "Enabled" if you want the menu item to exist but not display in the menu.
  7. In "Parent link" choose the parent of the new menu item.
  8. You can set the "Weight" here, or you can set it after saving by adjusting it using the drag-and-drop functionality.
  9. The "Power menu" allows you to sync menu items up with the Editorial Site Section. Under "Taxonomy plugin" find the term you just created (as per the instructions above) and select it. Most of the other terms will be grayed out, indicating they have already been paired with a menu item.
  10. Under most circumstances you will not need to add to or edit the rest of the fields.
  11. Click "Save" to save your new menu item.

Changing the placement of a main menu item

This is a two step process: 1) Adjust the Editorial Site Section term and 2) adjust the menu item itself.

  1. Change the site section term by going to Structure -> Taxonomy and then clicking on "list terms" for Editorial Site Section. Find the term you want to move/edit and use its crosshair icon to drag it to where you want and/or click on its "edit" link to change its attributes.
  2. Change the menu item placement and/or attributes by going to Structure -> Menus and then "list links" for the main menu. Find the menu item you want to move/edit and use its crosshair icon to drag it to where you want and/or click on its "edit" link to change its attributes.

How to edit a menu

  1. Using the admin toolbar, go to Structure -> Menus
  2. Find the menu you want to edit and click "list links." The available menus are:
    • Footer logos: the sister sites
      If you need to change the URLs the logos point to, change them here. If you need to change the logo, find the file in /sites/all/themes/omega_maricopa/images then replace the logo with one a similar size and with the same name.
    • Footer menu: the menu items in the footer
      Edit the text and paths of the menu items here. You can also rearrange their order by grabbing the crosshairs. Be sure to save after changing any of the ordering.
    • Main menu: the main navigation with the drop-downs
      You can edit the link text and paths here as well as change the ordering and nesting. Be sure to save after changing any of the placement.
    • Social media: the links for the social media icons
      If you need to change the URLs the icons point to, change them here. If you need to change the icon, find the file in /sites/all/themes/omega_maricopa/images then replace the icon with one the same size and with the same name.
  3. To change the properties of any of the links, such as the display title or link destination, click on its "edit" link.
  4. On the edit page for a menu item you can change the "Menu link title," the "Path," whether it is "Enabled" or not, its "Parent link," its "Weight," and its "Link to existing term" (which associates it with an Editorial Site Section).
  5. Be sure to click "Save" after making any changes.

What's the difference between the main menu and the site sections?

The main menu is what shows up in the top menu (with the drop-downs) and in the left sidebar. It gives visitors a structured way to explore the site.

The Editorial site section vocabulary does two things: 1) it allows for editorial access control and 2) page/content can latch onto a menu path without needing to actually be in the menu.

In some cases, you will not want every page to show up in the main menu structure, but you will still want it to fall under access control and to have the right breadcrumbs and sub-menu blocks to show up. In such instances you will not add it to the menu but will still give it a site section.
