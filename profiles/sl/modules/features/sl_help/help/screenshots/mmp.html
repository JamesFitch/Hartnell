Menu mini panels make additional sub-pages available from the main navigation. It provides a quicker route for visitors to find the pages they are looking for. Used carefully, megamenus can be a great enhancement to a site’s User Interface.

What do they look like?


Setting up Menu Mini panels

We are assuming that the module is downloaded and enabled, along with its dependencies.

  1. First, select Structure from the admin menu, then Mini Panels
  2. On the Mini panel page, click + Add.
  3. On the Add a new mini panel page, type in an Administrative title. Click Continue.
  4. The next page, Context, is beyond the scope of this documentation. Click Continue.
  5. The next page, Layout, you will want to explore a little. In the pulldown called Category, there are a number of different layouts. We recommend keeping it simple until you're used to filling minipanels -- choose a layout of one, two or three columns. And then click Continue.
  6. This last page, Content, is the most important one of all. You will want to click on the tiny, almost not visible cog in the top lefthand side of the Center box. For your enjoyment, I've blown up the little guy to 3 times its normal size.

    When you click on the cog, you will see a tiny menu -- and you will want to click on Add Content
  7. Since we are talking about Menu minipanels, when you come to the Add content to page, you will want to click Menus. And then Main Menu or whichever Menu you will be using. And then Finish.
  8. Do a Minipanel for each Menu item

Setting up the Menu

  1. Again, select Structure from the admin menu, then Menus.
  2. On the Menu page, pick Main Menu (or whichever Menu you will be using), and click list links.
  3. Click on + Add link to add a link.
  4. Add the title and the Path.
  5. Make sure the Parent link is <Main Menu> (or whatever Menu you will be using).
  6. Set the Weight (remember, lighter numbers rise [or go left] while heavier number sink [or go right]).
  7. On Menu Minipanel, choose the name of the minipanel you want to use.
  8. For the first time around, go with the default settings, and click Save.
  9. Do a Link for each Menu item

Editing a Menu Mini Panel

  1. First, select Structure from the admin menu, then Mini Panels
  2. From the list of existing Mini panels, pick the one you'd like to edit, and click Edit
  3. The navigation within this section is under the title Edit mini panel XXXX
  4. Settings and Context are not generally where you will want to make changes; but Layout and /or Content are.
  5. Obviously, Layout is for changing the layout — you will want to explore a little. In the pulldown called Category, there are a number of different layouts.
