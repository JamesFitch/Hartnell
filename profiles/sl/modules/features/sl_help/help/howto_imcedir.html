How to configure IMCE for browsing specific directories

IMCE is the file browser within the rich text editor. Enabling content creators to use the IMCE means they will be able to browse and select images on the server, upload images, create thumbnails and resize images, and much more.

  1. Go to the IMCE config page: &path&/admin/config/media/imce
  2. Click on the profile name you wish to edit, such as the "Content editors" profile
  3. Scroll down to the "Directories" section
  4. In a new entry, enter the directory you wish users to access, e.g. "imce/icons"
    Note: the root is going to be this path: &path&/sites/default/files So the full path will end up being something like: &path&/sites/default/files/imce/icons
  5. Apply the specific permissions you wish users to have, such as browsing, uploading, generating thumbnails, deleting files, and adding/removing subdirectories.
  6. If you have created that directory separately, make sure its permissions (and those of the files within) allow for Drupal to read and write to it.
