How to push your info to the General Numbers directory

As part of the the Community Connections site section there is a Directory page.

Where does this information come from?

All of the information you see on the Directory page originates from the Location content type. If you are unsure how to create the Location content type check out this tutorial.

  1. To successfully list on the Directory page you will need to complete the Title, Address and Phone Number fields on the Location content type
  2. To make sure it appears on this directory check the Published box and of course always click the Save button to preserve your changes.
