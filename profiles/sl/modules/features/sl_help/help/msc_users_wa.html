Workbench Access

As per the official module page, "Workbench Access creates editorial access controls based on hierarchies. [...] When creating and editing content, users will be asked to place the content in an editorial section. Other users within that section or its parents will be able to edit the content. A user may be granted editorial rights to a section specific to his account or by his assigned role on the site."

Setting up site section management

To set up specific users to manage particular sections of the site using Workbench Access:

  1. Using the admin menu, go to "Configuration" then scroll to the bottom to find "Workbench Access."
  2. Find the section over which the user should have content editing privileges. Note that if a section has subsections, the user will have editing privileges in all subsections.
  3. Click on the editors link corresponding to the section to which you want to grant administrative access.
  4. Type the name of the user in the Add editor autocomplete box.
  5. Click "Update editors."
