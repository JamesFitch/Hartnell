Changing the sister site images

The images for the sister site logos are stored in the theme directory, specifically: /sites/all/themes/omega_maricopa/images/ and are prefixed with "logo-."

Changing a logo

If you need to change a logo, save your new version with the exact same name as the old one, then upload it to the images directory, overwriting the old file.

Try to keep the dimensions approximately the same as the old version, to prevent any layout issues with the logos.

Adding a new logo

This is more complex than just adding a new logo image; you will also need to create a new menu item and corresponding styles. That is beyond the scope of this help documentation.
