Stlying a link to a menu

Before you begin

If the link you plan to add leads to a Drupal node or content type, first be sure your content type is published. Unpublished nodes may not be available to all site editors or visitors.

Let’s review how your site’s top navigation menus are organized.

  • Academic Style Pages filter first level links of the Main Menu or http://yourwebaddress.com/admin/structure/men through the top Navigation Bar.
  • Athletic Style Pages filter first level links of the Athletic Theme Menu or http://yourwebaddress.com/admin/structure/menu through the top Navigation Bar.

  1. Navigate to the content type for the menu (Structure > Menu or http://yourwebaddress.com/admin/structure/menu/types).
  2. Locate the menu you would like to edit (Main menu or Ahtletic Theme Menu) and click edit.
  3. Now for styling the Main menu be aware that these styles only apply to First Tier Menu links. These are primary links that you see a the top of all the pages. So it you refer to the Menu link list below, 'Hartnell College', 'A through Z index' and 'Future Students' are all First Tier Menu links. So you would not select Registration, or 'Orientation Calendar'.

Adding a link using the Menus section

  1. Navigate to the Menus page (Structure > Menus or http://yourwebaddress.com/admin/structure/menu).
  2. Locate a menu to edit and click add link.
  3. In the Menu link title field, enter the title as it will be displayed in the menu.
  4. In the Path field, enter a path for the link. This can be an internal path such as node/add (see "Before you begin" above) or an external URL such as http://example.com. Enter to link to the front page.
  5. In the Description field, enter the text that will be displayed when a user hovers over the link.
  6. Enable the link.
    • If the link is not enabled, it will not be displayed in the menu.
    • If the content is not published, it will not be displayed in the menu.
  7. In the Parent link list, select the parent of the link.
  8. (Optional) In the Weight list, select the relative weight of the link. Links with the "lightest" weight will display higher in a menu. Links with the same weight will display in alphabetical order.
  9. (Important) Now we ad the style class. Scroll down and under 'Menu Item Attributes' in the 'Classes' field you will add your resuable style for the top level navigation.
    • For top level links on the Academic Style Pages you will use the style 'main-menu-link '.
    • For top level links on the Academic Style Pages you will use the style 'athletic-link '.
  10. Click Save.

Adding a styled link using the content type

  1. Navigate to the node editing screen (Node > Edit or http://egos-school.com/admin/node/*/edit) for the node/content you want to add to the menu. This can also be done when you first create the node/content.
  2. Click the checkbox marked Provide a menu link. The menu options will appear.
  3. In the Menu link title field, enter the title as it will be displayed in the menu.
  4. In the Description field, enter the text that will be displayed when a user hovers over the link.
  5. In the Parent item list, choose the menu that should display this item. Note: the menu you want will only appear in this list if you have assigned it to the content type of the node you are editing. See "Before you begin" above.
  6. (Optional) In the Weight list, select the relative weight of the link.
  7. (Important) Now we ad the style class. Scroll down and under 'Menu Item Attributes' in the 'Classes' field you will add your resuable style for the top level navigation.
    • For top level links on the Academic Style Pages you will use the style 'main-menu-link '.
    • For top level links on the Academic Style Pages you will use the style 'athletic-link '.
  8. Click Save.
