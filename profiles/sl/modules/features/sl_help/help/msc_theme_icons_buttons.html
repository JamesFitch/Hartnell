Changing / adding to button icons

You can see a current list of available icons for use in buttons on the Reusable Styles help page.

The images for the icons are stored in the theme directory, specifically: /sites/all/themes/omega_maricopa/images/button-icons

Changing an icon

If you need to change an icon, save your new version with the exact same name as the old one, then upload it to the button-icons directory, overwriting the old file.

Try to keep the height approximately the same as the old version, to prevent any layout issues in the buttons.

Adding a new icon

This is more complex than just adding a new icon image; you will also need to create a new reusable style.

  1. Create your new icon image and upload it to the button-icons directory, using the same naming convention as the other icon files, e.g. "icon-customerservice.png".
  2. Open up the "button-icons" stylesheet. If you are using Compass/SASS (which is ideal), open "button-icons.scss" in the "sass" directory of the theme. If you are not familiar with Compass/SASS, you will also need to make your changes to the "button-icons.css" file in the "css" directory of the theme. However, you should not ignore the button-icons.scss file since if anyone ever recompiles the stylesheets your changes to the plain CSS file will be lost.
  3. In the stylesheet(s), copy the last rule and paste it at the bottom. Remember to do this in both files if you are not using Compass/SASS.
  4. Change the class name to what you want for your new button. To follow the preexisting pattern, use the text that follows icon- in the file name. For example, the "www" icon's file name is "icon-www.png" and its class name is "www." So if you were creating a new icon called "icon-flower.png" you would want to call the new class "flower."
  5. In the line that says "background-image" change the file name in the path to match the file name of the icon you just created and uploaded.
  6. Save the files.
  7. Test the new icon by creating a button and then adding the icon class.
  8. If you need to adjust the padding, open the SCSS (and the CSS file, if you're not using Compass/SASS) and edit the padding in rule you created.
